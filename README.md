# expo-sunmi-printer

React Native module for Sunmi Label Printer

## Installation


```sh
npm install expo-sunmi-printer
```


## Usage


```js
import { multiply } from 'expo-sunmi-printer';

// ...

const result = multiply(3, 7);
```
